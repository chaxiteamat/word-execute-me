# World.execute(me) - 交互式音乐可视化

一个基于Canvas和GSAP的交互式音乐可视化项目，灵感来自Mili的歌曲《World.execute(me)》。

## 🎵 项目简介

这是一个将音乐、代码和视觉艺术完美结合的交互式网页应用。通过Canvas动画和GSAP动画库，将《World.execute(me)》的歌词转化为震撼的视觉体验，展现了数字世界中代码执行的美感。

## ✨ 特性

- 🎨 **丰富的视觉效果** - 多种动画场景，包括粒子系统、3D投影、维度变换等
- 🎵 **音乐同步** - 动画与音乐节奏完美同步
- 🌟 **渐变与发光效果** - 精心设计的渐变和发光效果增强视觉冲击力
- 📱 **响应式设计** - 适配不同屏幕尺寸
- ⚡ **高性能** - 优化的Canvas渲染，流畅的动画体验

## 🎬 动画场景

项目包含多个精心设计的动画场景：

1. **开场** - 震撼的粒子系统开场
2. **代码世界** - 展示代码执行的视觉效果
3. **维度变换** - 从0D到4D的维度演进
4. **几何图形** - 圆周、几何变换等数学之美
5. **时空穿梭** - 时间轴与空间变换
6. **数字洪流** - 数据流动的视觉效果
7. **终章** - 完美的收尾动画

## 🛠️ 技术栈

- **HTML5 Canvas** - 主要渲染技术
- **GSAP (GreenSock Animation Platform)** - 动画库
- **JavaScript ES6+** - 核心逻辑实现
- **CSS3** - 样式与布局

## 🚀 快速开始

### 本地运行

1. 克隆项目到本地
```bash
git clone https://gitee.com/yourusername/world-execute-me.git
cd world-execute-me
```

2. 启动本地服务器
```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx http-server -p 8000
```

3. 在浏览器中打开
```
http://localhost:8000
```

### 直接访问

直接打开`index.html`文件即可在浏览器中查看效果。
